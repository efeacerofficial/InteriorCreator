A modern, fast, and user-friendly Windows Forms application designed to streamline the creation and management of Markers and Interiors for GTA / MTA (Multi Theft Auto) servers. Say goodbye to manual coordinate tracking and messy text files!

## ✨ Features

* 🌑 **Modern Dark Theme:** A sleek, eye-friendly, and professional user interface.
* 📍 **Precise Coordinate Management:** Easily manage `X, Y, Z` coordinates and `Player Angles` for both starting and destination markers.
* 🏢 **Interior Routing:** Link "From" and "To" interior IDs seamlessly.
* 📂 **File Export:** Select custom directory paths and export your configured marker lists directly to your project folders.
* 📋 **Real-Time List Tracking:** View, refresh, and clear your created markers in real-time within the application.
